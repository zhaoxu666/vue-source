# 数据驱动模型

Vue 执行流程
 +  获得模板：模板中有 '坑'
 +  利用 Vue 构造函数中所以工的数据来 '填坑'，得到可以在页面中显示的 '标签了'
 +  将标签替换页面中原来有坑的标签

Vue 利用 我们提供的数据 和页面中 模板 生成了一个新的 HMTL 标签 （node元素），
替换到了 页面中 放置模板的位置
